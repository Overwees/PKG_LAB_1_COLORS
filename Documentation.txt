Основа проекта - это:
1)colormath - вспомогательный класс, который по моей задумке будет хранить
значения цвета во всех цветовых моделях, функции перевода из одной
цветовой модели в другую , а также будет генерировать исключения при невозможности
перевода из 1-й модели в другую

Класс colormath пока что содержит только метод для задания цвета через RGB,
переменные значений цвета во всех цветовых моделях и 2 функции перевода
из RGB в CMYK и наоборот. Планируеться, что именно эти методы будут при задании
цвета через 1 любую цветовую модель подсчитавать значения цвета для других 
цветовых моделей и генерировать исключения
2)mainwindow - создание основного окна и его интерфейса
Примерный вид окна показан в window_screen.png

В верхней части окна можно выбрать цвет можно по клику на квадрат, показывающий цвет, по нажитии кнопки "Выберите цвет"
или введении значения в 16-тиричной системе в соответсвующее поле(Пока не реализовано). А также выбор моделей реализован
выпадающим списком виджета QComboBox (Пока не знаю сделать ли 1 с 18 вариантами или же 3 с выбором 1 из 6 цветовых моделей).
В нижней части окна будут отображаться 3 цветовые модели с возможностью ввода значений цвета или изменении их при
помощи ползунком.(пока не реализовано)

Список функций и их предназначение:
1)MainWindow(QWidget *parent) - основные настройка при запуске приложения	
2)void paintEvent(QPaintEvent*) - прорисовка цвета для квадрата
3)void darkTheme() - установка тёмной темы для приложения
4)void setDataInputLayout() - настройка основных Layout'ов для упорядочивания виджетов и их растяжении/сжатии
при изменениях размера окна

5)bool eventFilter(QObject *obj, QEvent *event) - определение клика на цветной квадрат и вызов диалогового окна
6)void ChooseColor() - Вызов диалогового окна выбора цвета при нажитии на кнопку "Выберите цвет" и клике на цветной квадрат
7)void ChooseModels(int i) - выбор отображения цветовой модели через QComboBox
8)void groupBoxesSettings() - заготовка QGroupBox'ов для всех 6 моделей, пока имеються только поля ввода(планирую реализовать ползунки)
